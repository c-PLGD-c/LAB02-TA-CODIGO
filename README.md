# Predictive Maintenance Prototype using ESP32

## Description
This repository contains the source code for the ESP32-based predictive maintenance prototype presented in the IEEE paper.

## Hardware
- ESP32 DevKit
- QMC5883L Magnetometer
- Electromagnetic Coil
- IRLZ44N MOSFET
- 1N4007 Diode

## Software
- Arduino IDE
- ESP32 Core
